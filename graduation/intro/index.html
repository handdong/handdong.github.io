Numerical Optimization Intro

术语

minf(x),$x\in$$R^{n}$,并满足一些约束条件st

st:
ci(x)=0, i$\in$E
ci(x)>=0, i$\in$I

$x\in$$R^{n}$ - 变量
f(x)$\in$R - 目标函数
ci(x)=0, i$\in$E - 等式约束函数
ci(x)>=0, i$\in$I - 不等式约束函数
$\Omega$=$\left { x: ci(x)=0, i \in E, ci(x)>=0, i \in I\right }$ - 可行域
$x^{\star}$ - 问题的解(要满足约束条件和最小的要求)

用途

  1. 运输问题

m个仓库 a1 a2 … am >= 0
n个工厂 b1 b2 … bn >= 0
xij: 从第i个工厂运送xij顿钢材到第j个工厂
cij: 从第i个工厂运送1顿钢材到第j个工厂的费用
min$\sum_{i}$$\sum_{j}$cij * xij
st:
$\sum_{j}$xij<=ai, 1<=i<=m
$\sum_{i}$xij>=bj, 1<=j<=n

  1. 机器学习

子类

  1. 无约束优化 VS 约束优化
  2. 线性规划,目标函数、约束函数都是线性的
  3. 二次规划,目标函数是二次函数、约束函数是线性函数
  4. 凸规划,目标函数是凸的、约束函数是线代函数

有点关系的概念

  1. 局部和全局
  2. 连续和离散
  3. 决定与随机

关于“数值”

在算法设计时，要考虑舍入误差

基本内容

  1. 解的存在、唯一、特征等
  2. 算法（迭代算法、收敛性、收敛速度）
